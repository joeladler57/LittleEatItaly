# Little Eat Italy - PRD

## Original Problem Statement
Build a modern website for "Little Eat Italy" Neapolitan pizzeria with urban, drippy, red/black/white graffiti style aesthetic. Full backend and frontend. No online ordering, no reservation system.

## User Personas
- **Pizza Lovers**: Looking for authentic Neapolitan pizza
- **Local Customers**: Neighborhood residents seeking quality dining
- **Urban/Trendy Demographic**: Attracted by the street art aesthetic

## Core Requirements (Static)
- Modern graffiti/urban aesthetic with red/black/white palette
- Home page with hero section and featured items
- Menu page with category filtering
- About page with restaurant story
- Contact page with form
- Animations and smooth transitions
- Mobile responsive design

## What's Been Implemented (Feb 17, 2026)
- ✅ Full-stack application (React + FastAPI + MongoDB)
- ✅ Urban graffiti design with Anton + Space Mono fonts
- ✅ Home page with hero, features, featured menu, CTA sections
- ✅ Menu page with category tabs (All, Classic, Special, Vegetarian, Dessert)
- ✅ About page with story, stats, philosophy sections
- ✅ Contact page with form and location info
- ✅ Backend APIs: /api/menu, /api/menu/featured, /api/contact, /api/settings
- ✅ Data seeding with 10 menu items
- ✅ Framer Motion animations throughout
- ✅ Noise overlay texture effect
- ✅ Drip border effects
- ✅ Glass-morphism cards
